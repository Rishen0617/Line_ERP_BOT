# Google Sheets 欄位規格

## 同一個 Spreadsheet，4 個 Sheet

---

## Sheet1：收據記錄

| 欄 | 名稱 | 說明 | 格式 |
|---|---|---|---|
| A | 日期 | 文件日期 | YYYY-MM-DD |
| B | 上傳時間 | 系統時間 | YYYY-MM-DD HH:MM |
| C | 上傳者 | LINE 顯示名稱 | 文字 |
| D | 文件類型 | 收據/發票/出貨單/其他 | 文字 |
| E | 廠商/客戶 | 廠商名稱或客戶名稱 | 文字 |
| F | 品項摘要 | 前 3 項，頓號分隔 | 文字 |
| G | 金額 | 不含貨幣符號 | 數字 |
| H | 稅額 | | 數字 |
| I | 單號 | **唯一值，重複不寫入** | 文字 |
| J | 圖片連結 | Google Drive webViewLink | URL |
| K | AI 信心度 | 高/中/低 | 文字 |
| L | 群組 | LINE group_id | 文字 |
| M | 備註 | 人工補充欄 | 文字 |

---

## Sheet2：記帳流水帳

| 欄 | 名稱 | 說明 |
|---|---|---|
| A | 日期 | YYYY-MM-DD |
| B | 類別 | 收入/支出/應付/應收 |
| C | 說明 | 交易說明 |
| D | 金額 | 數字 |
| E | 對象 | 廠商或客戶 |
| F | 關聯單號 | 對應 Sheet1 欄 I |
| G | 記錄者 | LINE 顯示名稱 |

---

## Sheet3：訂單追蹤

| 欄 | 名稱 | 說明 |
|---|---|---|
| A | 訂單日期 | YYYY-MM-DD |
| B | 供應商 | |
| C | 品項摘要 | |
| D | 金額 | |
| E | 狀態 | 待確認/已確認/出貨中/已到貨 |
| F | 物流單號 | |
| G | 建立者 | |

---

## Sheet4：月報（公式自動計算，程式不寫入）

使用 SUMIF 從 Sheet2 彙總各月收支：

```excel
=SUMIFS(記帳流水帳!D:D, 記帳流水帳!A:A, ">="&DATE(B1,C1,1), 記帳流水帳!A:A, "<"&DATE(B1,C1+1,1), 記帳流水帳!B:B, "收入")
```
